# 抽芽 🌱

轻量化桌面自律日程与待办管理工具。

# 安装
> Python 3.12+ 推荐，Windows / macOS 均可运行。

# 安装依赖（仅需托盘最小化功能）
pip install pystray Pillow

## 快速开始

```bash
python main.py
```

## 功能概览

### 自律日程
- **三类周期**：记日、记周、记月，每种独立刷新
- 目标次数（1~127），每次点击 `＋` 递增，达标后自动标记完成并在月历留痕
- **标签分类**：创建彩色标签，事项按标签分组显示
- **计时专注**：事项可设定倒计时（1~180 分钟），开始后进入专注模式，到期弹窗提醒
- **最大完成次数**：达到指定次数后自动归档为"永久完成"
- 点击月历格子查看当日完成记录与待办截止

### 待办记事
- 标题、描述、截止日期、提醒时间
- 重要-紧急四象限散点图 + 亟待解决时间线图
- 到达提醒时间自动弹窗通知

### 工作台
- 左侧自律事项 + 右侧亟待解决待办，两侧均支持快速完成
- 专注计时模式：🪴 盆栽 + "努力生长中" + 倒计时，期间锁定页面

### 通用
- 浅色 / 深色主题切换
- JSON 格式导入导出数据
- 关闭窗口最小化到系统托盘，提醒继续后台运行

## 项目结构

```
抽芽/
├── main.py              # 入口
├── config.py            # 全局常量
├── theme.py             # 浅/深主题定义
├── app/
│   ├── root.py          # 主窗口 & 应用逻辑
│   ├── sidebar.py       # 侧边栏导航
│   ├── content.py       # 内容区页面容器
│   ├── statusbar.py     # 状态栏
│   └── pages/
│       ├── schedule.py  # 自律日程
│       ├── todo.py      # 待办记事
│       ├── workbench.py # 工作台
│       └── settings.py  # 设置
├── data/
│   ├── database.py      # SQLite 数据管理
│   └── settings_manager.py  # 用户偏好
└── utils/
    ├── helpers.py
    └── thread_pool.py
```

## 数据存储

所有数据存储在 `C:/Users/~/.抽芽/data.db`（SQLite），偏好设置存储在 `~/.抽芽/preferences.json`。
